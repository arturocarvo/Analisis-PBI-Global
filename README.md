# Analisis-PBI-Global
Este proyecto de Data Science se enfocó en el análisis de indicadores económicos globales (PBI, PBI per cápita, composición sectorial) para identificar tendencias y evaluar la predictibilidad del crecimiento económico futuro.

Puntos Clave:

Análisis Exploratorio: Se confirmó la divergencia de crecimiento entre economías emergentes y desarrolladas, y la transición sectorial como clave del desarrollo económico

Modelado Predictivo: Se aplicó Feature Selection (Random Forest) para identificar las 10 features más influyentes.
